# Telegram-бот для IT-услуг

Бот умеет:

- показывать услуги;
- показывать пакеты;
- принимать заявку на консультацию;
- принимать обращение в поддержку 24/7;
- показывать контакты;
- отправлять заявки админу в Telegram;
- отправлять заявки на почту.

## 1. Создать бота в Telegram

1. Откройте Telegram.
2. Найдите `@BotFather`.
3. Напишите:

```text
/newbot
```

4. Укажите название бота.
5. Укажите username бота, например:

```text
office_it_support_bot
```

6. BotFather выдаст токен вида:

```text
123456789:AA....
```

Скопируйте его.

## 2. Узнать свой Telegram ID

1. В Telegram найдите бота:

```text
@userinfobot
```

2. Нажмите Start.
3. Скопируйте свой ID.

## 3. Настроить .env

Скопируйте файл:

```bat
copy .env.example .env
```

Вставьте:

```text
TELEGRAM_BOT_TOKEN=токен_из_BotFather
ADMIN_TELEGRAM_ID=ваш_Telegram_ID
```

Почта для заявок уже указана:

```text
LEAD_EMAIL_TO=galolo066@gmail.com
```

Если хотите отправлять заявки на почту через Gmail, нужен App Password от Gmail.

## 4. Запуск бота

Откройте CMD в папке `telegram-bot` и выполните:

```bat
npm install
npm start
```

Если всё хорошо, будет:

```text
Telegram-бот запущен.
```

## 5. Как остановить

В окне CMD нажмите:

```text
Ctrl + C
```

## 6. Важно

Пока окно CMD открыто — бот работает.
Если закрыть CMD — бот остановится.

Чтобы бот работал постоянно, его лучше запускать на VPS/VDS сервере или через PM2/NSSM.
